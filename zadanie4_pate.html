1)kiedy powstał JavaScript (podaj datę).
JavaScript został stworzony w 1995 roku, a oficjalnie zaprezentowany w grudniu 1995.

2)dlaczego jest to język skryptowy
Bo kod JS nie jest kompilowany do pliku wykonywalnego jak np. C++, tylko jest interpretowany „w locie” przez przeglądarkę (lub silnik JS na serwerze).

3)gdzie jest wykonywany JS (klient lub serwer)
Tradycyjnie po stronie klienta (w przeglądarce), ale dzięki Node.js może być też wykonywany po stronie serwera.

4)dlaczego jest JS to bezpieczne narzędzie
Bo JS w przeglądarce działa w tzw. piaskownicy (sandbox), czyli ma ograniczony dostęp do systemu operacyjnego np. nie może kasować plików na dysku użytkownika.

5)czy jest to język obiektowy , jeśli tak to dlaczego.
Tak, JavaScript jest językiem obiektowym, bo korzysta z obiektów, dziedziczenia (prototypowego) i pozwala tworzyć własne klasy/obiekty.

10)Przepisz linie kody pod nimi wytłumaczenie jak działa ten fragment kodu
< input type="button" name="przycisk" value="Nowa Strona" onclick="WinOpen(' ')"> Tworzy przycisk o nazwie „przycisk” i napisie „Nowa Strona”, a po jego kliknięciu wywołuje funkcję WinOpen(' '), która ma otworzyć nowe okno/stronę.

11)Przepisz linie kody pod nimi wytłumaczenie jak działa ta ten fragment kodu
window.open("obraz.html","okienko","toolbar=no,directories=no,menubar=no,height=280,width=160,top=200,left=200"); Otwiera nowe okno przeglądarki z plikiem obraz.html, nadaje mu nazwę „okienko” i ustawia określone parametry wyglądu oraz położenia. b)toolbar=no - ukrywa pasek narzędzi

c)directories=no - ukrywa katalogi

d)menubar=no - ukrywa menu

e)height=280 - wysokość okna 280 px

f)width=160 - szerokość okna 160 px

g)top=200 - odległość od góry 200 px

h)left=200 - odległość od lewej 200 px

12)Opisz następującą instrukcję window.close()
Polecenie window.close() zamyka bieżące okno przeglądarki lub okno otwarte wcześniej za pomocą window.open().
